• Use the left and right arrow keys to move the piece.
  • Use the S and A keys to rotate the piece.
  • Use the down arrow to drop the piece.
I originally coded this project in Racket for one of my freshman courses. I rewrote it in Java outside of class the next semester, which allowed me to use the 2D Graphics libraries as well as use some of the object oriented design topics I had learned. Adding the high score feature allowed me to experiment with networking and object serialization. Check out the code and javadoc documentation under the tetris menu heading.
